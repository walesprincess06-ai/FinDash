# FinDash - Personal Finance Tracker

A full-stack application designed to help users track their liquidity, monitor monthly budgets, and visualize their spending habits in real-time. 

## Key Features
* **Secure Authentication:** User login and registration powered by Firebase Authentication (Email/Password & Google OAuth).
* **Real-Time Database:** Uses Cloud Firestore with strict security rules to ensure user data privacy and instant UI updates without page reloads.
* **Interactive Analytics:** Integrates Chart.js for dynamic cash flow doughnut charts and 6-month savings trend visualizers.
* **Smart Budgeting:** Tracks category-specific spending against user-defined limits with automated visual alerts.
* **Modern UI:** A clean, responsive, glassmorphism-inspired dashboard styled with Tailwind CSS and Lucide Icons.

## Tech Stack
* **Frontend:** HTML5, Vanilla JavaScript (ES6+), Tailwind CSS
* **Backend:** Firebase (Auth, Cloud Firestore)
* **Libraries:** Chart.js, Lucide Icons
